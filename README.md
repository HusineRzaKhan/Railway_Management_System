This simple project is created in Java language with Object Oriented Programming (OOP) technique;

It has following features;
     1. Login/Signup as Customer
     2. Menu for User
     3. Book Tickets
     4. Passenger Details
     5. Search about Train Details
     6. Station Details
     7. Java GUI for FrontEnd
